# AnxEase - Smart System for Diagnosing Anxiety Disorders

## Overview

AnxEase is an AI-powered mental health support system designed to assist users in assessing anxiety levels, tracking mood patterns, and receiving intelligent insights through a user-friendly mobile application.

This project was developed as a Graduation Project for the Bachelor of Computer Science and Information.

## Features

* User Authentication & Authorization
* Anxiety Assessment System
* Mood Tracking
* Intelligent Analysis
* Personalized User Profiles
* Interactive Question Flow
* Secure Data Management

## Technologies Used

### Frontend

### Backend

* Python
* Flask

* SQLite
